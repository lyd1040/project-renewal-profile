# 프로젝트 실행
## node_modules 설치
```
npm install
```

### 프로젝트 실행
```
npm run serve
```
### 프로젝트 실행시 주의사항
```
배포때문에 코드 경로 관련된 주소들 앞에 /renewalprofile를 붙였기 때문에
설치후 실행시 경로에 /renewalprofile 는 지우고 실행시켜야 함
```

## 사용 기술스택

-  Vue
-  Typescript
